# Esti Search Grid - Volunteer Quick Guide

Open the map here:

https://esti-search-grid.vercel.app

This map helps the team see which 1 km search squares are being covered. It works on phones and updates for everyone.

## Before You Start

1. Open the link on your phone.
2. Enter your name, phone number, and team if you have one.
3. Tap **Save identity**.
4. If your phone asks for location permission, allow it if you are comfortable. GPS helps show which grid you are near.

## Claim A Grid

1. Find the square you are searching.
2. Tap the square.
3. Tap **Keep searching**.
4. The grid is now assigned to you so others can see it is being searched.

If a square is already assigned to someone else, choose another square or contact the dispatcher.

## While Searching

- Keep the page open if possible.
- Tap your grid again and use **Send heartbeat** every 15 to 20 minutes.
- If there is no heartbeat for 30 minutes, the system may mark the grid stale so another team can check it.

## When You Are Done With A Grid

Tap the grid again and choose one:

- **Search complete**: you finished checking that square.
- **Stopped**: you are leaving that square before completing it.
- **Need backup**: you need another volunteer or team to help.
- **Emergency**: urgent safety issue. Call 911 or command first, then mark it.
- **Found Esti**: if you believe you found Esti, call command immediately. Call 911 first if there is any urgent safety or medical concern. Then mark it.
- **Keep searching**: you are still actively searching that square.

## GPS

Tap **Locate me** to show your approximate location on the map. GPS can be wrong by several meters, especially near buildings, so use it as a guide, not as proof that you are inside a specific square.

## If The App Looks Wrong

- Refresh the page.
- Check that it says **Shared sync: live**.
- If it says offline, keep searching and tell the dispatcher your grid number by phone/text.

## Important Safety Notes

- Do not enter unsafe areas alone.
- Do not rely on the app instead of calling for help.
- For emergencies, call 911 or the search lead first.
- Do not put private or sensitive details in notes unless command asks you to.

Every grid action is logged with time, user, grid, and action type so command can reconstruct the search timeline.
